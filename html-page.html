Introduction to HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a webpage, using tags to define elements.

How it's Used

HTML is used to define the structure of a web page by specifying headings, paragraphs, lists, images, links, and more. It allows web developers to organize and present information in a structured and meaningful way.

What it Does

HTML defines the content of a web page, making it accessible and readable by web browsers. It allows users to interact with web content, navigate through pages, and access information on the internet. HTML forms the foundation of the World Wide Web.

Example 1: Basic HTML

<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Page</h1>
    <p>This is a paragraph of text.</p>
  </body>
</html>
