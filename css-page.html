What is CSS?

CSS (Cascading Style Sheets) is a language used to control the presentation and styling of web documents, including HTML and XML documents. It defines how elements should appear on a web page, such as layout, colors, fonts, and spacing.

How CSS is Used

CSS is used by web developers and designers to separate the structure (HTML) from the presentation (CSS) of a web page. It allows for consistent styling across multiple pages, making websites visually appealing and user-friendly.

Examples of CSS

Here are some common use cases for CSS:

  • Text Styling: Change font size, color, and style.
    p { color: blue; font-size: 18px; }
  • Layout Control: Adjust the positioning and size of elements.
    .box { margin-left: 20px; padding: 10px; }
  • Backgrounds: Add background colors or images.
    body { background-color: #f86e6e; }
  • Responsive Design: Make websites adapt to different screen sizes.
    @media screen and (max-width: 768px) { /* styles for smaller screens */ }
